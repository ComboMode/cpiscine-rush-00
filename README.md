# cpiscine-rush-00

Para fazer os testes adicione os `#include` corretos no `main.c`, por exemplo:

```c
...

#include "ft_putchar.c"
#include "rush0X.c"

...
```

Feito isto, é só compilar o `main.c` e executar o seu resultado da seguinte maneira:

```shell
$ gcc main.c -o a.out
$ chmod +x a.out
$ ./a.out
...
```

Pronto! :)
